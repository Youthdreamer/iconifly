# Iconifly React

一个基于 React 的图标组件库。

## 简介

图形图标组件库，

## 安装

使用 npm 安装:

```bash
npm install @iconifly-react
```

或使用 yarn:

```bash
yarn add @iconifly-react
```

## 使用方法

```jsx
import { Homeicon } from '@iconifly/react/icons';
import { Filmgraphic } from '@iconifly/react/graphics';
```
